# Online-Bookstore

## Description

You are required to design and develop a fully functional Online Bookstore Web 
Page using HTML, CSS, and JavaScript. This project evaluates your ability to 
integrate structure, styling, and interactivity into a cohesive web application. 
The project is an extended version of the mid-term task , with enhanced requirements 
suitable for group collaboration.
